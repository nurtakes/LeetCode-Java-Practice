# ☕ LeetCode Java Practice

Bu repository, Java programlama dilindeki temel yeteneklerimi tazelemek, sentaks pratikleri yapmak ve algoritmik düşünme becerimi güçlendirmek amacıyla çözdüğüm LeetCode problemlerini içermektedir.

## 🎯 Hedeflerim
* Nesne Yönelimli Programlama (OOP) konseptlerini Java'da aktif olarak kullanmak.
* Temel Veri Yapıları (Diziler, Bağlı Listeler, Hash Map vb.) üzerinde pratik yapmak.
* Problem çözme pratiği kazanarak temiz ve optimize edilmiş kodlar yazmak.

## ⚙️ Kullanılan Teknolojiler
* **Programlama Dili:** Java
* **Platform:** LeetCode
